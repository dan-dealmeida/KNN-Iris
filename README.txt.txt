Deve-se existir 2 arquivos .txt, um com 4 features e um labeled (labeled dataset). E outro arquivo .txt com 4 features e sem label (unlabeled dataset).

O arquivo do labeled dataset deve estar no formato especificado:
exemplo:
5.1,3.5,1.4,0.2,Iris-setosa

1) representa largura da petala
2) representa altura da petala

3) representa largura da sepala
4) representa altura da sepala
5) label: setosa, versicolor ou virginica

Cada linha contendo os features de um único objeto.



O arquivo do labeled data set deve estar no formato especificado:
exemplo:
5.1,3.5,1.4,0.2

1) representa largura da petala
2) representa altura da petala

3) representa largura da sepala
4) representa altura da sepala

Cada linha contendo os features de um único objeto.

Apos a execucao deve-se escolher primeir oa pasta contendo o labeled dataset e depois a pasta contendo o unlabeled dataset.
